# 🧠 Mind Match - Memory Card Matching Game

A premium and interactive Memory Card Matching Game built with React, TypeScript, and Vite. The game challenges players to match card pairs while tracking moves, time, accuracy, and streaks. It also includes an AI Memory Buddy, sound effects, leaderboard tracking, and multiple difficulty levels.

## 🎯 How to Play

1. Start the game by selecting a difficulty level.
2. Flip cards to reveal their symbols.
3. Match identical card pairs.
4. Matched cards remain visible.
5. Unmatched cards flip back automatically.
6. Complete all matches with the fewest moves and best accuracy.
